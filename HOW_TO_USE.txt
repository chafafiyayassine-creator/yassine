==========================================
HOW TO USE YOUR PROFESSIONAL WEBSITE
==========================================

STEP 1: VIEW YOUR WEBSITE
--------------------------
1. Find the file called "index.html" in your folder
2. Double-click on "index.html"
3. It will open in your web browser (Chrome, Firefox, Edge, etc.)
4. You will see your professional website!

That's it! The website is ready to use.

STEP 2: CUSTOMIZE YOUR WEBSITE
-------------------------------

A. CHANGE THE PRODUCT NAME:
   1. Open "index.html" in a text editor (like Notepad or VS Code)
   2. Press Ctrl+F to search
   3. Search for "ProductName" (without quotes)
   4. Replace it with YOUR product name
   5. Save the file (Ctrl+S)
   6. Refresh your browser to see changes

B. CHANGE THE TEXT/CONTENT:
   1. Open "index.html"
   2. Find the text you want to change
   3. Edit the text between the tags (like <h1>YOUR TEXT HERE</h1>)
   4. Save the file
   5. Refresh your browser

C. CHANGE COLORS:
   1. Open "style.css"
   2. Find the section that starts with ":root {"
   3. Change the color codes (like #6366f1 to #ff0000 for red)
   4. Save the file
   5. Refresh your browser

STEP 3: UPDATE CONTACT INFORMATION
-----------------------------------
1. Open "index.html"
2. Find the section with id="contact"
3. Change the address, phone, and email
4. Save and refresh

STEP 4: UPDATE FEATURES
------------------------
1. Open "index.html"
2. Find the section with id="features"
3. Change the feature titles and descriptions
4. Save and refresh

QUICK REFERENCE:
----------------
- index.html = Main website file (edit text here)
- style.css = Design and colors (edit colors here)
- script.js = Interactive features (usually don't need to edit)
- HOW_TO_USE.txt = This file (instructions)

TIP: After making changes, always save the file and refresh your browser (F5) to see the changes!

NEED HELP?
----------
- Make sure all 3 files (index.html, style.css, script.js) are in the same folder
- If something doesn't work, check that you saved the file after editing
- Refresh your browser (F5) to see changes
